# Random Number Guessing Game

A simple terminal-based number guessing game written in C.

The program generates a random number between 1 and 100 and the user keeps guessing until the correct number is found. The program gives hints if the guess is too high or too low, and on succesful guess do tell how many attempts wew taken by the user.

## Concepts used
- rand() and srand()
- loops
- conditional statements
- user input
